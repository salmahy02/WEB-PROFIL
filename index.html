Web development refers to the creating, building, and maintaining of websites. It includes aspects such as web design, web publishing, web programming, and database management. It is the creation of an application that works over the internet i.e. websites.

UI/UX Design

UI UX designers create the user interface for an app, website, or other interactive media. UX Design refers to the term User Experience Design, while UI Design stands for User Interface Design. Both elements are crucial to a product and work closely together.

Internet Of Things

The Internet of Things (IoT) describes the network of physical objects—“things”—that are embedded with sensors, software, and other technologies for the purpose of connecting and exchanging data with other devices and systems over the internet.

Database

A database is an organized collection of structured information, or data, typically stored electronically in a computer system. Computer databases typically store aggregations of data records or files that contain information, such as product information.

Smart Farming: Bubble Smart Plant

The project created is a Smart Farming "Bubble Smart Plant: Watering Control and Monitoring System Automatic Planting of Chili Plants Using Wemos D1".
